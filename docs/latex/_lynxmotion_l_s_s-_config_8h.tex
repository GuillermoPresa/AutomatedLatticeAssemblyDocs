Enable this option to log all bus transactions to a Serial port. by default we log to the first serial port \textquotesingle{}Serial\textquotesingle{} which all Arduino\textquotesingle{}s have, but you can change this to any Arduino stream supporting class. 

Generate a pulse on a pin when bus read errors occur. If defined and set to a GPIO pin, the library will toggle this pin anytime a read error has occured. This is great for tracing electrical errors using an oscilloscope. Hook the GPIO pin to the trigger input of your oscope.

Tracks statistics on bus transactions (NOT YET IMPLEMENTED) Statistics will include bus transaction counts, timing, read errors, and servo contention.
