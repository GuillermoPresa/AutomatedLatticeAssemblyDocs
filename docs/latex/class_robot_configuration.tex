\doxysubsection{Detailed Description}
Robot configuration is the full representation of a robot position. It consists on a \mbox{\hyperlink{class_pos_j}{PosJ}} and a Baselocation. Realize that \mbox{\hyperlink{class_pos_j}{PosJ}} just represents a configuration of the tool with respect to the base but in robots where the base can also move a full description must provide the base position. The base position is stored in the form of the Node the base is clamped to.
