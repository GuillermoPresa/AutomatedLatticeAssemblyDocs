Planning on the graph\+:

\begin{DoxyVerb} Planning is the reasoning side of acting.\end{DoxyVerb}


Utilizing the dynamic programming principle based on Bellman\textquotesingle{}s principle of optimality\+:

\begin{DoxyVerb} An optimal policy has the property that whatever the initial state and initial decision are, the remaining decisions must constitute an optimal policy with regard to the
 state resulting from the first decision.\end{DoxyVerb}


{\bfseries{Process\+:}}

At any point in the search, we can classify the states into 3 groups; Unvisited, dead or alive ~\newline
 {\itshape Unvisited\+:} states that have not been visited yet ~\newline
 {\itshape Dead\+:} States that have been visited togeather with every possible next state ~\newline
 {\itshape Alive\+:} State that has been visited but whose possible next states have yet to be visited ~\newline
 {\bfseries{Notation\+:}} ~\newline
 States are denoted by the letter X.

X\+: states ~\newline
 X\+\_\+goal ~\newline
 X\+\_\+initial ~\newline


{\bfseries{Search algorithms\+:}} ~\newline
 f(x) = g(x) + e h(x) ~\newline
 f(x)\+: Cost utilized for evaluated the priority of the priority queue. Deternubes which nodes will be visited next. ~\newline
 g(x)\+: Cost of the path from the starting state x\+\_\+\{i\} to the current state\+\_\+\{x\} ~\newline
 h(x)\+: Heuristic function that aproximates the cost from the current state x to the goal state x\+\_\+\{g\} ~\newline
