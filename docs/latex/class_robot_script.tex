\doxysubsection{Detailed Description}
\hypertarget{class_robot_script_RobotScriptMain}{}\doxysubsection{Robot Script}\label{class_robot_script_RobotScriptMain}
Robot script and state machine manager.

A high quality introduction to state machines and description of the particular implementation can be found in \href{https://www.youtube.com/watch?v=Vt8aZDPzRjI&list=PLwyUzJb_FNeTR1Q7edAQuWkTKo_Ncq9ck&index=2}{\texttt{ YT video State machine}}

This state machine implementation does not allow for custom state switching that is different for every transtion but rather general enter and exit states functions are provided.

State machine with states\+: -\/Waiting, Y -\/Yielding, N -\/Moving, Y -\/Inserting, N -\/Loading N -\/Fault N
