\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_pos_j}{PosJ}} represents the Pose of a robot in the Joint configuration, that is the rotation of all the joints. \mbox{\hyperlink{class_pos_j}{PosJ}} depends on the Hierarchy of the robot. It represents the join in the following order\+: 1) Base 2) Shoulder 3) Middle 4) Wrist 5) Gripper. If the hierarchy is inverted the new current and target Pos\+Js will have the inverted order.
