\doxysubsection{Detailed Description}
This class contains methods that must be run from inside a thread and others that must be invoked from Unity. Both types of methods are clearly marked in the code, although you, the final user of this library, don\textquotesingle{}t need to even open this file unless you are introducing incompatibilities for upcoming versions.
