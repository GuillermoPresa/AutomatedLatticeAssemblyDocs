\hypertarget{index_intro_sec}{}\doxysection{Introduction to automated lattice assembly}\label{index_intro_sec}
The project consists on the automated assembly of discrete lattice structures via walking robots in the structure as showcased in the image. The program presented aims to simplify the process of discretizing build geometry into voxels, controlling the robot, path planning and decision making.

\begin{DoxyWarning}{Warning}

\begin{DoxyPrecond}{Precondition}
Install a supported version of unity engine \href{https://unity.com/}{\texttt{ https\+://unity.\+com/}}
\end{DoxyPrecond}
The program employs Unity as a 3D rendering and physics platform. The UI or user interface is built using Unity Engine tools, that can not be ported to standalone builds thus a working version of the engine needs to be installed for operating the program.
