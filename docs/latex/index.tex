\hypertarget{index_intro_sec}{}\doxysection{Introduction to automated lattice assembly}\label{index_intro_sec}
The project consists on the automated assembly of discrete lattice structures via walking robots in the structure as showcased in the image. The program presented aims to simplify the process of discretizing build geometry into voxels, controlling the robot, path planning and decision making.

\hypertarget{index_install_sec}{}\doxysection{Installation}\label{index_install_sec}
Written in C\#, the software utilizes Unity engine for 3D visualization, operations and user interface. The user interface is based on IMGUI, a simple code driver UI toolkit designed for custom inspectors. The application is to be run within the editor and not intended to be run in the form of a standalone execturable for the time being.
