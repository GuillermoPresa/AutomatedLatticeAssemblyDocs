\doxysubsection{Detailed Description}
Class that represents the possible ways to transition from one node to another. When traversing or moving from one lattice node (base) to another (target), there is a set of possible joint configurations we can take to perform this motion. This is represented by the movement class.

This class consists of a starting robot configuration (starting \mbox{\hyperlink{class_pos_j}{PosJ}} and base location), the target \mbox{\hyperlink{class_pos_x}{PosX}}, and a list of possible Pos\+Js to reach the target.

Movements are constructed from example by the inverse kinnematics or I.\+K functions of a robot controller.

At the time of executing the movement one of the target Pos\+Js can be selected according to a cost calculation.
