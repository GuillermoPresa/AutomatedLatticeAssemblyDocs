\doxysubsection{Detailed Description}
While \textquotesingle{}\mbox{\hyperlink{class_serial_controller}{Serial\+Controller}}\textquotesingle{} only allows reading/sending text data that is terminated by new-\/lines, this class allows reading/sending messages using a binary protocol where each message is separated from the next by a 1-\/char delimiter.
