\doxysubsection{Detailed Description}
\hypertarget{class_illa_navigation_path_Illa}{}\doxysubsection{navigation graph class}\label{class_illa_navigation_path_Illa}
The ILLA navigation graph class allows to send and comunicate navigation paths that can easily be inverted and operated in a simple manner.

It consists of a list of gameobjects that represent the nodes of the graph. movement dictionary that stores the optimal movements to perform where the key of the dictionary is the node of the graph you are on and the value shows the way to reach the next.
