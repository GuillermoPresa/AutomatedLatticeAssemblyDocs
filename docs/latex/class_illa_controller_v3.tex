Before the servo motors are enabled, the motor microcontroller is active and reporting its position but compliant with the motor unpowered. At this point the target configuration is being updated to the current configuration as moved by the user, in order to prevent a big displacement when connection is stablished.

Once connection is stablished the motors become fully active. The target configuration is no longer updated to match the current configuration but rather it can be updated via the user interface leading to robot movements to match the current position to the target position.\hypertarget{class_illa_controller_v3_autotoc_md12}{}\doxysubsubsection{Background knowledge}\label{class_illa_controller_v3_autotoc_md12}
