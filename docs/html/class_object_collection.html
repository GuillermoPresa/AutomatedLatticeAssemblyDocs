Lattice collections

Lattice collections are the representation of the group of lattices in the selected group of objects to be built. They contain information about what lattices are placed, what lattice is to be placed next, as well as including built in querries for finding common lattice sets such as placed, unplaced etc... in the build order. Note that the build order can differ from what is stablished in the inut structure definition due to impossible routing or additional constraints.
