Detailed Description

Main governing script of ILLA inchworm locomotion lattice assemblers.

Note
Integrates the concepts from PosJ, PosX, RobotConfiguration and Movement.
Warning
Illa is based on Inverse Kinnematics and high flexibility. It has the capability of performing collision estimation. Some of these calls can however be very computationally expensive.
