bool CollisionDetectionScene.CheckCollisionPermanentPrefabs ( GameObject  CollisionPrefab,
List< GameObject >  LatticeList 
)

Check for collisions between a list of game objects and a collision prefab.

This function checks for collisions between a list of game objects (LatticeList) and a collision prefab (CollisionPrefab). It splits the LatticeList into smaller batches and checks for collisions between the collision prefab and each batch using the CheckCollisionBatch function. If any collisions are detected, it returns true. Otherwise, it returns false.

Parameters
CollisionPrefabThe collision prefab to use for the collision check.
LatticeListThe list of game objects to check for collisions with.
Returns
True if any collisions are detected, false otherwise.
