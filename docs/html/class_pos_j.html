Detailed Description

A PosJ represents the Pose of a robot in the Joint configuration, that is the rotation of all the joints. PosJ depends on the Hierarchy of the robot. It represents the join in the following order: 1) Base 2) Shoulder 3) Middle 4) Wrist 5) Gripper. If the hierarchy is inverted the new current and target PosJs will have the inverted order.
