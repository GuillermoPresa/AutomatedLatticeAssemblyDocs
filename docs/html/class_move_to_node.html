Action to move the robot to a far away node. During the setup of this action it is further broken into smaller actions. It allows some commonality in the breaking of tasks into actions as many actions require movements like this. A series of parameters allow for some flexibility in the breaking of the action into smaller actions.

Parameters
targetNode_in
moveToNeighbor_inIf true, the robot will move to the closest neighbor of the target node instead on the target node directly.
