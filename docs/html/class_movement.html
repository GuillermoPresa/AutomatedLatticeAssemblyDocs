Detailed Description

A movement is a class that represents a particular robotic movement. It consists of a starting robot configuration (starting PosJ and base location), a target PosX, and a list of possible PosJs to reach the target. At the time of executing the movement one of the target PosJs can be selected according to a cost calculation.
