Detailed Description

Robot Script

Robot script and state machine manager.

A high quality introduction to state machines and description of the particular implementation can be found in YT video State machine

This state machine implementation does not allow for custom state switching that is different for every transtion but rather general enter and exit states functions are provided.

State machine with states: -Waiting, Y -Yielding, N -Moving, Y -Inserting, N -Loading N -Fault N

Robot defines a work volume
