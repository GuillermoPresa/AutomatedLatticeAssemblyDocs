Detailed Description

Illa node sensor. Utilized to sense the presence of nodes that can be gripped by the base and the gripper.

Warning
Do NOT paralellice
Do NOT access its properties directly. In order to prevent issues with the collision detection, the collider of this node sensor is inactive until the moment of checking. Requires the processing of a physics frame.
